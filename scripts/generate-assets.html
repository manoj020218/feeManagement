FeeFlow — Asset Generator

Generates all Android icons, adaptive layers, Play Store icon, and splash screens. Open in Chrome → Download All.

How to use

  1. Click Download All Icons → extract the zip (each file is named for its mipmap folder)
  2. Copy each PNG to android/app/src/main/res/mipmap-*/ic_launcher.png and ic_launcher_round.png
  3. Copy adaptive foreground → mipmap-anydpi-v26/ic_launcher_foreground.png
  4. Copy splash PNGs to drawable-port-*/splash.png (portrait) and drawable-land-*/splash.png (landscape)
  5. Play Store 512×512 → upload to Google Play Console under Store listing → App icon
